# 🍔 Zomato Food Delivery Dashboard (Power BI Project)

This project is a complete data analysis and visualization dashboard based on Zomato food delivery data. It provides meaningful insights into customer behavior, food trends, transactions, and delivery performance.

---

## 📊 Project Overview

The dashboard helps to analyze:

- Total Orders & Quantity
- Monthly Sales Trends
- Food Item Popularity
- Transactions by Payment Method
- Delivery Status (Delivered vs Cancelled)
- Customer Type (Gold vs Regular)

---

## 🚀 Key Features

✅ Interactive Dashboard with Filters  
✅ Monthly Performance Analysis  
✅ Food-wise Sales Insights  
✅ Payment Method Distribution (UPI, COD, Card)  
✅ Customer Segmentation (Gold vs Regular)  
✅ Delivery Success Rate  

---

## 📈 Insights from Dashboard

- 📌 Most ordered food item: **Samosa**
- 📌 Highest transactions via: **UPI**
- 📌 Maximum orders in: **February & March**
- 📌 Delivery success rate: **94%+**
- 📌 Main Course category dominates sales
